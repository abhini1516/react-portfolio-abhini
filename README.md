### **Prerequisites**

Ensure you have Node.js (v18 or higher) and npm installed on your machine.

### **Installation & Setup**

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/react-portfolio-abhini.git
    ```

